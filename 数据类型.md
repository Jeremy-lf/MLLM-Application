## 2.结构化数据与非结构化数据
所谓结构化输出，指的是模型输出结果以一种规则化、预定义的形式呈现，通常包括表格、列表、树形结构、JSON 格式等。结构化输出具有明确的层次和逻辑关系，易于解析和操作。非结构化输出则指的是自由文本、自然语言形式的回答，这类输出尽管在可读性上较强，但往往缺乏预定义的结构，处理起来更加复杂。

结构化数据
```
{
  "patient_id": 12345,
  "diagnosis": "type_2_diabetes",
  "severity": "moderate",
  "recommendation": [
    "increase_physical_activity",
    "dietary_adjustments"
  ]
}
```

